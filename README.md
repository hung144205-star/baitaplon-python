# Quản Lý Dịch Vụ Cho Thuê Kho Lưu Trữ Hàng Hóa

## Giới thiệu
Phần mềm quản lý dịch vụ cho thuê kho lưu trữ hàng hóa - Nhóm 12

## Cấu trúc project
```
.
├── src/              # Mã nguồn chính
├── docs/             # Tài liệu
├── tests/            # Unit tests
├── data/             # Dữ liệu (database, logs)
├── config/           # File cấu hình
└── README.md         # Hướng dẫn tổng quan
```

## Yêu cầu hệ thống
- Python 3.8+
- SQLite

## Cài đặt
```bash
pip install -r requirements.txt
```

## Chạy chương trình
```bash
python src/main.py
```
