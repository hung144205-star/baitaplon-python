# Yêu cầu chức năng

## 1. Quản lý khách hàng
- Thêm mới khách hàng
- Cập nhật thông tin khách hàng
- Xóa khách hàng
- Tìm kiếm khách hàng

## 2. Quản lý kho hàng
- Thêm kho mới
- Cập nhật thông tin kho
- Theo dõi sức chứa kho
- Báo cáo tình trạng kho

## 3. Quản lý hợp đồng thuê
- Tạo hợp đồng thuê kho
- Gia hạn hợp đồng
- Chấm dứt hợp đồng
- Thanh toán

## 4. Quản lý hàng hóa
- Nhập hàng vào kho
- Xuất hàng khỏi kho
- Theo dõi tồn kho
- Kiểm kê định kỳ

## 5. Báo cáo thống kê
- Doanh thu theo thời gian
- Tỷ lệ lấp đầy kho
- Hợp đồng sắp hết hạn
- Khách hàng tiềm năng
